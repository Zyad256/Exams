# 🎓 Basmaja

**Basmaja** is a modern, dynamic educational platform designed to help students prepare for exams with speed and efficiency. Featuring a sleek **"Neon Glassmorphism"** aesthetics interface, multi-modal exam environments, and deep post-exam statistical analysis.

## ✨ Features

- 📱 **Mobile-First Responsive UI**: Flawless layout adapting instantly to desktop, tablet, and smartphone screens.
- 🎨 **Glassmorphism Design System**: Modern translucent overlays, gradient accents, and curated google typography.
- 🧠 **Dynamic Subject Context**: Seamlessly switches between diverse engineering subjects using a globally hoisted state-engine.
- ⏱️ **Real-Time Exam Simulator**: Supports deep-dive exam modes, mock practice, timed rounds, flag mechanisms, and randomized arrays.
- 📊 **Intelligent Analytics**: Immediate post-exam feedback tracking exact accuracy percentages, question duration mapping, and robust theoretical explanations for incorrect responses.
- 🎥 **Embedded Google Integrations**: Natively renders Google Slides presentations and lectures directly in the course hub alongside rapid per-lecture quizzes.

## 📚 Curriculum Hub

Basmaja proudly features dynamic routing for several high-level subjects.

### 1. Computer Vision

- Covers Image Processing, Human Vision Systems (HVS), Edge Detection, and Feature Extraction.
- Features **11 Audio-Driven Lectures** natively embedded into the dashboard.
- Backed by **3 Core Midterm Exams** and an automated system of **9 Per-Lecture Quizzes** pinned directly below their respective lecture presentations.
- 45+ unique validated questions with deep-theory explanations.

### 2. Advanced Web Programming

- Covers distributed systems, LSM Trees, SSTables, and Database Replication architecture.
- 3 Dedicated Exams simulating full term and midterm difficulties.

## 🚀 Architecture & Tech Stack

This project is built from the ground up prioritizing speed and native flexibility.

- **Core State**: Vanilla React (Vite) context providers for centralized dynamic subject switching.
- **Design System**: Global custom CSS parameters circumventing heavy frameworks.
- **History Tracking**: Safe localized progression storage arrays enabling offline stat accumulation.
